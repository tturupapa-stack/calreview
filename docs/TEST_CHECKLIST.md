## 9. 테스트 실행 방법

### 자동화 테스트 실행
```bash
# 모든 테스트 실행
npm test

# Watch 모드
npm run test:watch

# 커버리지 리포트
npm run test:coverage
```

### 수동 테스트 환경 설정
1. 환경 변수 설정 (`.env.local`)
2. 개발 서버 실행: `npm run dev`
3. 테스트 계정 준비 (네이버, Google)
4. Supabase 데이터베이스 연결 확인
