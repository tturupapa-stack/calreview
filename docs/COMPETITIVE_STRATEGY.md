## 1. 경쟁사 분석

### 1.1 콜라보매니저 (주요 경쟁사)

| 항목 | 내용 |
|-----|------|
| 운영사 | 주식회사 아마고 |
| 서비스 형태 | 모바일 앱 (iOS/Android) |
| 핵심 기능 | 체험단 정보 모음 + 스케줄러 + 정산 관리 |
| 수집 사이트 | 아싸뷰, 레뷰, 강남맛집, 리뷰플레이스, 리뷰노트, 서울오빠, 가보자체험단, 미블, 디너의여왕 |
| 비즈니스 모델 | 무료 (추정: 광고/프리미엄) |

#### 강점
- 앱스토어 정식 등록 (신뢰도)
- 체험단 사이트 제휴 관계
- 스케줄러 + 정산 관리 올인원
- 시장 선점 효과

#### 약점
- 앱 설치 필요 (진입장벽)
- 개인 관리 도구에 집중 → "발견"보다 "관리"
- 검색/필터링 기능 제한적
- 커뮤니티/정보 공유 부재
- 선정 확률 등 인사이트 없음

## 2. 캘리뷰 현황

### 2.1 서비스 개요

| 항목 | 내용 |
|-----|------|
| 서비스 형태 | 웹 (Next.js) |
| 핵심 기능 | 체험단 통합 검색/비교 |
| 수집 사이트 | stylec, modan, chuble, dinodan, real_review (5개) |
| 현재 캠페인 수 | ~2,400개 |

### 2.2 기술 스택
- Frontend: Next.js 14, React, Tailwind CSS
- Backend: Supabase (PostgreSQL)
- Crawler: Python (BeautifulSoup)
- Deployment: Vercel

## 3. 포지셔닝 전략

### 3.1 핵심 차별화

```
콜라보매니저: "체험단 관리 도구" (After)
캘리뷰: "체험단 발견 플랫폼" (Before)
```

### 3.2 사용자 여정 분석

```
[발견] → [비교] → [신청] → [관리] → [리뷰작성] → [정산]
   ↑                              ↑
 캘리뷰                       콜라보매니저
```

**전략: 사용자 여정의 "입구"를 장악하여 주도권 확보**

## 4. 이기는 전략 5가지

### 4.1 전략 1: "선정 확률" 데이터 제공

> 💡 핵심 Pain Point: "신청해도 떨어지는데 어디가 잘 되나요?"

#### 기능 상세
- [ ] 캠페인별 예상 경쟁률 표시
- [ ] "이 조건의 블로거는 선정률 72%"
- [ ] 사이트별 평균 선정률 비교
- [ ] 나와 비슷한 블로거의 선정 이력

#### 데이터 수집 방안
- 사용자 제보 기반 (선정/탈락 결과 입력)
- 모집인원 vs 실제 선정인원 추적

#### 우선순위: ⭐⭐⭐ (중기)

---

### 4.2 전략 2: 웹 기반 + PWA (설치 없이 즉시 사용)

> 💡 핵심: 앱 설치 = 이탈

#### 기능 상세
- [x] 웹에서 바로 검색/필터링
- [ ] PWA 지원 (홈화면 추가)
- [ ] 카카오/네이버 소셜 로그인
- [ ] 30초 만에 시작 가능한 UX

#### 마케팅 메시지
"설치 없이 30초 만에 체험단 찾기"

#### 우선순위: ⭐⭐⭐⭐ (단기)

---

### 4.3 전략 3: AI 맞춤 추천

> 💡 "내 블로그에 맞는 체험단"

#### 기능 상세
- [ ] 블로그 URL 입력 → 카테고리/방문자 분석
- [ ] "당신의 블로그에 선정 확률 높은 캠페인"
- [ ] 마감 임박 + 경쟁률 낮은 캠페인 알림
- [ ] 주간 맞춤 추천 이메일/카톡

#### 기술 요구사항
- 네이버 블로그 분석 API 또는 크롤링
- 추천 알고리즘 개발
- 알림톡 연동

#### 우선순위: ⭐⭐⭐ (중기)

---

### 4.4 전략 4: 체험단 리뷰어 커뮤니티

> 💡 콜라보매니저에 없는 "소셜" 요소

#### 기능 상세
- [ ] 사이트별 실제 후기/평점
- [ ] "이 업체 정산 잘 해주나요?" 정보
- [ ] 초보자 가이드/팁 공유
- [ ] 선배 리뷰어 멘토링

#### 기대 효과
- 정보 비대칭 해소
- 충성 유저 확보
- 자연스러운 바이럴

#### 우선순위: ⭐⭐ (장기)

---

### 4.5 전략 5: 리뷰 마감 리마인더 (무료 핵심 기능)

> 💡 체험단 최대 고민: "리뷰 마감일 놓침"

#### 기능 상세
- [ ] 카카오톡/SMS 리마인더
- [ ] 체험일 → 리뷰 마감일 자동 계산
- [ ] 캘린더 연동 (구글, 애플)
- [ ] 무료 제공 → 바이럴 유도

#### 기술 요구사항
- 카카오 알림톡 API
- 구글/애플 캘린더 연동
- 푸시 알림 시스템

#### 우선순위: ⭐⭐⭐⭐ (단기)

## 7. 브랜딩

### 7.1 슬로건 후보

1. "체험단, 찾지 말고 발견하세요"
2. "매일 1,000개 체험단, 당신에게 맞는 건 3개"
3. "선정률 높은 체험단만 모았습니다"
4. "체험단 검색의 새로운 기준"

### 7.2 핵심 가치

- **시간 절약**: 여러 사이트 돌아다닐 필요 없음
- **인사이트**: 단순 정보가 아닌 분석 제공
- **맞춤화**: 내 블로그에 맞는 추천
- **커뮤니티**: 리뷰어들의 정보 공유

## 8. 리스크 및 대응

### 8.1 법적 리스크

| 리스크 | 대응 방안 |
|-------|---------|
| 크롤링 차단 | robots.txt 준수, 요청 빈도 제한 |

| 리스크 | 대응 방안 |
|-------|---------|
| 콜라보매니저 기능 확장 | 차별화된 인사이트 기능 강화 |
| 신규 경쟁자 진입 | 커뮤니티로 락인 효과 확보 |

## 9. 제휴 전략

### 9.1 잠재 제휴처

| 구분 | 대상 | 제휴 내용 |
|-----|-----|---------|
| 체험단 사이트 | 레뷰, 리뷰노트, 서울오빠 등 | 공식 API/데이터 제공 |
| 블로그 플랫폼 | 네이버 블로그 | 블로그 분석 연동 |
| 마케팅 | 인플루언서 | 서비스 홍보 |

### 9.2 제휴 접근 전략

1. 트래픽 유입 데이터로 가치 증명
2. 상호 이익 구조 제안
3. 작은 사이트부터 시작 → 레퍼런스 확보
